# V-ICU | Virtual Cardiac ICU Monitor

Симулятор моніторингу пацієнтів після кардіохірургічних операцій у реальному часі.

## Стек

- React 19 + TypeScript
- Vite
- TailwindCSS v4
- Recharts
- Motion/React

## Запуск локально

**Вимоги:** Node.js 18+

```bash
npm install
npm run dev
```

Відкрийте [http://localhost:3000](http://localhost:3000)

## Збірка

```bash
npm run build
```

Результат у папці `dist/`.
